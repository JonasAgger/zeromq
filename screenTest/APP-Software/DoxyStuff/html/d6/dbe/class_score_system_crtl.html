osapi::Message *  msgPtr 
)

--------—— HANDLE PSOC UPDATE ---------—— -— HANDLES THE VALUES RETURNED FROM THE PSOC -—

Handles the updates received from the psocs, by casting the Message pointer to a psocUpdateMessage, where an uint8_t value can be retrived from. It will then discard the update, if its received during rearrange, which means, that no team can finish, while 1 is rearranging. It will also ready the game, when initially setting up the cups, making it possible to press play game.

Otherwise it checks if the current state is of less value than the newly received state, which will cause a change of state. This happens untill a cupZoneReady state is received.

The teams scores are also updated in this function.
